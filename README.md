# FormCheck
Библиотека для валидации и проверки форм в TypeScript/JavaScript.

## Установка

```bash
npm install formcheck
# или
yarn add formcheck
# или
pnpm add formcheck
```

## Использование 

```bash
import * as f from "formcheck"
```
