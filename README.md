# FormCheck
Библиотека для валидации и проверки форм в TypeScript/JavaScript.

## Установка

```bash
npm install formcheck
# или
yarn add formcheck
# или
pnpm add formcheck
```

## Использование 

```bash
import * as f from "formcheck"
```

## Дополнительно

Для корректной работы файла tsconfig.json, используйте команду 

```bash
npm install --save-dev vite
```

Для проверки тестов, используйте команду 

```bash
npm run test -- --coverage
```
